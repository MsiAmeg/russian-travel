## Описание:
Третий проект на курсе веб-разработчика от Яндекс.Практикум.
В работе использованы инструменты адаптивной и "резиновой" верстки &rarr;
1. Flex/grid.
2. Медиа запросы.
3. Относительные единицы.

При разработке проекта применена технология БЭМ.
